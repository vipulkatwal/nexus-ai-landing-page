## Getting Started 🏁

### Prerequisites
- Node.js
- npm or yarn

### Installation 🔧
1. Clone the repository
```bash
git clone https://github.com/vipulkatwal/nexus-ai-landing-page.git
```

2. Install dependencies
```bash
cd nexus-ai-landing-page
npm install
```

3. Run the development server
```bash
npm start
```
